Week 6 Array

Array : กลุ่มของตัวแปรที่มีประเภทเดียวกัน ตัวแปรแต่ละตัวมี index กำกับอยู่ มีการกำหนดขนาดของอาร์เรย์ตั้งแต่แรก ชื่อของอาร์เรย์ใช้บอกตำแหน่งเริ่มต้นของอาร์เรย์นั้นได้
  - ขนาดของอาร์เรย์ควรเป็นค่าคงที่
  - สามารถใช้ #define directive หรือ const ช่วย
  - สามารถเก็บค่าอาร์เรย์หลายมิติได้แต่ใน memory จะเก็บเป็นอาร์เรย์แบบ 1 มิติ
1) #define : เป็นคำสั่งให้ compiler เปลี่ยนอักษรทั้งหมดในโปรแกรมก่อน compile
2) const : เป็นการประกาศตัวแปรให้เป็นค่าคงที่เฉพาะ block การทำงานนั้นๆ
  - pass by value : copy ค่าของตัวแปรจากผู้เรียกไปให้ฟังก์ชัน มีการสร้างตัวแปรใหม่ใน local variable และการเปลี่ยนค่าในฟังก์ชันไม่ส่งผลถึงตัวแปรอันเดิม
  - pass by reference : ส่งตำแหน่งของตัวแปรไปให้ฟังก์ชัน ตัวแปรเป็นตัวเดียวกับตัวแปรเดิมถ้าเปลี่ยนตัวแปรเดิมก็เปลี่ยน ในภาษาซีไม่มีแบบนี้แต่จำลองได้โดยการใช้ pointer
